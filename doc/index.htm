Rextile

Rextile 1.0 released!
Oct 20, 2006

Build XHTML documents and entire web sites using Textile—a concise format—and Ruby scripting.

  • Usage
  • Standard Template
  • Scripting
  • Editing

Rextile was inspired by Xilize. It uses RedCloth to convert Textile to XHTML, erb to run script blocks, and Hpricot for DOM manipulation.
